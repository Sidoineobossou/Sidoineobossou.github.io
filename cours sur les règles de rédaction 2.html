Cours sur les règles de rédaction

Salut à vous chers internautes. Je me présente. Mon nom est OBOSSOU Sidoine et je suis rédactrice web Seo.

Méthodologie de la rédaction web seo

Méta Description : comme vous l’entendez, elle décrit le contenu bref de votre article. Elle doit faire 130 à 135 caractères, espace compris.

Méta Titre/Title : Elle est aussi un titre exemplaire au h1. Respectez surtout la forme d’écriture de Méta Tire/Title.

NB : si on ne vous demande pas la Méta Description et le Méta Titre/Title dans la consigne, ne le faites pas. De plus, sachez aussi que parmi le nombre de mots qui vous a été demandé, la Méta description et le Méta titre/Title puis le H1 ne sont pas comptés.

Méthodologie respective.

H1 : c’est un titre

Introduction ou chapeau : Elle se fait au plus en 03 phrases. Mais évitez les annonces du plan sous forme interrogative. Toutefois vous pouvez mettre en guise d’exemple : Découvrez à la suite les surprises que vous réserve cet article.

h2 : c’est un titre
Paragraphes
Les paragraphes ne dépassent jamais 05 lignes. Si vous avez quelque chose à ajouter, allez à la ligne mais sachez articuler vos syntaxes.

h3 : c’est un titre
Paragraphes
Vous pouvez faire uniquement de H2. En fait les H2 sont les éléments du H1 à détailler, autrement dit, les sous-titres de H1 Les H3 sont les sous éléments des H2 à détailler. A vous d’établir votre plan si on ne vous exige aucun plan à suivre.

Conseils :
On ne met jamais H1 mais juste le titre. En fait c’est un code que les rédacteurs utilisent pour surnommer les titres.
Règles :
Pas de phrases longues : sujet verbe complément ; chaque phrase doit apporter un truc nouveau ; suivez les consignes surtout et ne faites pas de plagia ; étiez le hors sujet, c’est important. Vérifiez le plagia sur le site : 1 text
Vérifiez vos fautes inconscientes sur le site : Google docs.

Insertion de lien
L’encre est le mot du lien. Elle doit être bien choisie si on ne vous donne pas d’encre. L’ancre doit être le seul mot figurant dans votre article. Faites CTRL pour vérifier les mots clefs utilisés et votre ancre.
Ensuite cliquez sur votre ancre pour insérer le lien. Mais comment ? Cliquez insertion en Haut, vous verrez “lien“ en bas de là où est écrit Dites-nous ce que vous voulez faire. Ensuite copiez le lien qui vous a été envoyé et cliquez “enregistrer“ ou “ok“.
